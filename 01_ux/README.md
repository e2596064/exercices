# Exercice : Conception d'expérience utilisateur·rice

1. En équipe de trois ou quatre, choisissez un site Web que l'un·e de
   vous utilise fréquemment. Pour la durée de l'exercice, cette personne
   prendra le rôle de l'utilisateur·rice. Les autres membres de l'équipe
   seront des concepteur·rices d'expérience.
2. Identifiez un problème que l'utilisateur·rice a avec le site Web.
   Votre projet pour cet exercice est de régler ce problème.
3. Créez un document FigJam commun pour toute l'équipe. Sauvegardez les
   étapes suivantes dans ce document.
4. Formulez le problème sous la forme d'un [énoncé], et identifiez les
   [objectifs] du projet.
5. Demandez à l'utilisateur·rice de partager son écran, et prenez
   quelques minutes pour [observer] l'utilisateur·rice interagir avec le
   site Web. Notez vos observations.
6. Faites une [analyse des tâches]. Créez ensuite un
   [parcours utilisateur·rice].
7. Concevez un [persona].
