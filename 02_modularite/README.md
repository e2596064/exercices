# Exercice : Modularité

Pour cet exercice, vous devez reproduire dans Figma la [page d'accueil]
du site Web du Gouvernement du Québec. Commencez par définir une grille,
puis identifiez les différents composants de la page. Une fois les
composants définis dans Figma, assemblez le tout avec la disposition
automatique. Votre maquette doit utiliser les mêmes polices de
caractères, les mêmes couleurs, et respecter les zones de remplissage
(_padding_) et les marges. Utilisez l'inspecteur de votre navigateur
pour obtenir les bonnes valeurs.

[page d'accueil]: https://www.quebec.ca/
